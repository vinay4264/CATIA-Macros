# CATIA Macro – Part Auto Renamer

## 📌 Description
This macro automatically renames all components in a CATIA assembly using a structured naming format.

## ⚙️ Features
- Sequential renaming of parts
- Improves assembly organization
- Reduces manual effort

## 🛠️ Tools Used
- CATIA V5
- VBScript

## ▶️ How to Use
1. Open CATIA Assembly (.CATProduct)
2. Go to Tools → Macro → Run
3. Select macro file
4. Execute

## 🚀 Example Output
Part1 → VINAY_001  
Part2 → VINAY_002  

## ⚠️ Note
This macro is intended to run inside CATIA V5 environment.
